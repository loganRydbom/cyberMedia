# cyberMedia
A "social media" platform created with Vite and ChakraUI.

## Cloning
To clone, run the command:
```powershell
git clone https://github.com/loganRydbom/cyberMedia.git
```

## Node.js
To install packages, run:
```powershell
npm install
```
To run the development server, run:
```powershell
npm run dev
```
